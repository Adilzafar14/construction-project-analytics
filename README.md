## 📌 Project Overview
An end-to-end data analytics project analyzing **1,300 construction tasks** to identify cost patterns, risk levels, delay trends, and resource utilization — built to simulate real-world engineering project analytics in domains like **Assystem Radicon**.

---

## 🎯 Business Problem
Construction companies face challenges in:
- Identifying which risk levels drive the most cost
- Understanding task delay patterns
- Optimizing resource utilization across projects

---

## 📊 Key Findings
| Metric | Value |
|--------|-------|
| Total Tasks Analyzed | 1,300 |
| Total Project Cost | $64.38M |
| Average Delay | 14.77 days |
| Low Risk Cost Share | 51% |
| High Risk Avg Delay | 15.63 days |

## 💡 Insights
1. **Low Risk** tasks contribute 51% of total cost — highest budget allocation
2. **High Risk** tasks show maximum delays (15.63 days avg)
3. **High Resource Utilization** tasks cost more but complete faster
4. **Cost Variance Analysis** revealed T478 as highest budget overrun task
